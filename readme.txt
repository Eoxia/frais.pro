== Description ==

The plugin frais.pro allows you to fill your expense notes very easily.
1) Take the expense note in photo
2) Enter the amounts paid, VAT
3) Validate your expense report per person per month
4) Export directly to * .csv for your accountant
5) Archive in a "libreoffice.org" file for state controls

== Installation ==

The installation of the plugin can be done in two ways:

* Method 1
1. Download the zip file from the WordPress website
2. Send the "fresh" folder to the `/ wp-content / plugins /` directory
3. Activate the plugin in the "Extensions" menu of WordPress

* Method 2
1. Search the plugin "frais.pro" from the menu "Extensions" of WordPress
2. Start the installation of the plugin
